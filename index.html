Italiano

DocuSift funziona localmente sul dispositivo. Non richiede account, non invia documenti a server esterni e non usa servizi cloud per l'analisi. Anche la classificazione AI avviene interamente sul PC tramite un modello locale: nessun contenuto dei documenti lascia il dispositivo.

Dati trattati localmente:

  • percorsi file selezionati dall'utente;
  • metadati di analisi;
  • categoria, data, nome suggerito e cartella proposta;
  • hash file;
  • testo estratto solo se l'utente abilita l'opzione relativa.

Acquisti: l'eventuale acquisto della versione Pro avviene tramite Microsoft Store ed è gestito da Microsoft secondo la sua informativa privacy. L'app verifica soltanto lo stato della licenza tramite le API dello Store; nessun dato dei documenti è coinvolto.

L'utente può cancellare l'indice locale e la cronologia dall'app con Cancella tutti i dati locali. Le operazioni sui file vengono applicate solo dopo conferma. Il comportamento predefinito non modifica mai gli originali.

English

DocuSift runs locally on the device. It does not require an account, does not upload documents to external servers, and does not use cloud services for analysis. AI classification also happens entirely on the PC through a local model: no document content ever leaves the device.

Local data:

  • file paths selected by the user;
  • analysis metadata;
  • category, date, suggested name, and suggested folder;
  • file hash;
  • extracted text only when the user enables that option.

Purchases: the optional Pro upgrade is handled by the Microsoft Store under Microsoft's privacy statement. The app only checks the license state through Store APIs; no document data is involved.

The user can clear the local index and history from the app with Clear all local data. File operations are applied only after confirmation. By default, originals are never modified.
